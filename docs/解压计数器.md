# 解压计数器 · 极简无字设计（修正版三）

> **核心原则**：界面无文字，仅凭数字、图标与反馈完成所有交互。  
> **归零按钮定位**：首先是功能明确的归零工具，尺寸小于绿色主按钮，但保留高品质触感反馈，成为一个低调却上瘾的“小玩点”。

---

### 1. 全局布局

**屏幕方向**：竖屏，沉浸式深/浅背景。

**结构（从上到下）**：

- **顶部区域**：居中巨大计数器数字。
    - 数字**正右侧**紧挨一个**较小的圆形归零按钮**（直径约为绿色按钮的50-60%）。
    - 数字**右上角**放置极简日历图标（统计入口）。
- **中部主交互区**：水平滑动切换三组计数组件，一次完整显示一个。
- **底部指示器**：三个极简图标圆点，可点击切换。
    - ● 按压
    - ⌇ 滚动
    - ↕ 拉拽

---

### 2. 归零按钮（数字右侧·小型玩点）

**尺寸原则**：  
比绿色大按钮小约一半，保持足够的点击热区（≥40pt），但视觉上明显是“二级按钮”，不喧宾夺主。

**外观设计**：

- 小圆形，暖橙红渐变，表面硅胶高光质感。
- 中心图形为一个极简**逆时针回转箭头**，用细白线勾勒，无填充。
- 按钮微微凸起，投下细腻阴影，暗示可按压。

**交互反馈（核心玩点）**：

1. **按下瞬间**：
    - 按钮弹性缩小约10%，阴影加深，下压感明确。
    - 触发一次清脆短促的“嗒”声（与绿色按钮音调略有不同，偏高，带一丝电子感）。
    - 震动：`vibrateShort`，强度`light`。

2. **弹起瞬间**：
    - 按钮弹回原状，伴随微弱抖动回弹。

3. **归零事件**：
    - 数字归零，伴随一段短小的“沙沙”消散音效（如沙子流走，或旧式秒表归零的机械声）。
    - 震动：一次`heavy`长震（约30-40ms），与音效同步。
    - 数字动画：数字快速缩小至消失点，再从零重新弹出，带有轻微果冻弹性。

4. **连续点击**：
    - 每一次都立即归零，毫无延迟或阻拦。
    - 连续点击时，数字会形成“跳跃归零”的节奏感，配上反复的沙沙声与震动，成为一种令人上瘾的小动作。

> **设计意图**：  
> 它第一眼是“归零工具”，但一旦你按下它，那一刻的清脆音效、震动与数字消失动画，会让你忍不住再按一次。它小，所以你可以像拨弄一个开关、按一个钢笔按钮一样反复把玩，成为一种隐蔽的解压手段。

---

### 3. 按压模式（绿色大圆形按钮）

**图标指示器**：● 实心圆点  
**组件尺寸**：大型，占据中部区域主要视觉重心，直径约160-180px。  
**外观**：嫩绿到翠绿渐变，径向高光模拟软胶/硅胶质感，中心可留空或极微弱的“+”压痕。  
**交互反馈**：

- 点按计数+1。
- 按下：按钮缩放至95%，阴影收拢。
- 弹起：弹性回弹，轻微抖动。
- 音效：清脆“咔嗒”声（类似机械键盘青轴或戳破泡泡纸），音调每次微调+/-5%以抗疲劳。
- 震动：`vibrateShort`，强度`light`。
- 数字动画：数字轻微果冻弹跳。

> 它与归零按钮共享相同的手感基因，但音效和震动风格区分明显，一个用于“累积”，一个用于“清零”。

---

### 4. 滚动模式（刻度滚轮）

**图标指示器**：⌇ 三条竖线  
**组件形态**：横向圆柱形滚轮，金属磨砂质感，表面等距刻度线。中心上方有固定三角指示标。  
**交互逻辑**：

1. 手指拨动滚轮，滚轮跟随转动。
2. 施加**速度上限**，即使用力猛扒也不会飞转，保持舒缓节奏。
3. 手指离开后滚轮惯性滚动并逐渐减速至停止。
4. 每经过一个刻度线，计数+1，触发：
    - 棘轮“咔”声。
    - 震动`vibrateShort`，强度`medium`。
    - 滚轮边缘短暂高亮。
5. 静止时自动吸附至最近的刻度线，不会停在中间。

**解压点**：拨弄它，看着它缓缓停下，每一步刻度都有明确的反馈。

---

### 5. 拉拽模式（弹力拉带）

**图标指示器**：↕ 垂直短线与下拉箭头  
**组件形态**：竖向厚实橡胶带，亮黄或珊瑚色，带纵向防滑纹理。带上等距标记4个刻度圆点。底部固定，顶部有扩大的握持区（拉环造型）。  
**交互逻辑**：

1. 按住握持区下拉，带子拉伸。
2. 每经过一个刻度圆点，计数+1，触发：
    - 震动逐级递增（light → medium → heavy → heavier）。
    - 音效从“绷紧皮筋”到“极限拉伸”。
    - 带子表面高亮。
3. 拉过第4个刻度后**完全锁死**，边缘闪红，无法再拉。
4. 松手瞬间：
    - 带子极速弹回原位。
    - 响亮“啪”的一声。
    - 一次`heavy`长震动。
    - 整个界面轻微上下颤抖。
5. 可立即再次拉扯，无冷却时间。

**解压点**：暴力拉扯被硬性限制住，最后松手那一下的释放感极度舒适。

---

### 6. 声音与震动系统

| 动作         | 音效       | 震动            | 备注               |
| ------------ | ---------- | --------------- | ------------------ |
| 绿色按钮按下 | 清脆“咔嗒” | `light`         | 音调微随机         |
| 归零按钮按下 | 电子感“嗒” | `light`         | 比绿色按钮音调稍高 |
| 归零触发     | 沙沙消散音 | `heavy` 30-40ms | 与数字动画同步     |
| 滚轮刻度经过 | 棘轮“咔”   | `medium`        | 稳定节奏           |
| 拉带经过刻度 | 皮筋绷紧声 | 逐格递增        | 第4格最强          |
| 拉带弹回     | 响亮“啪”   | `heavy` 长震    | 界面微颤           |
| 切换组件     | 轻柔滑音   | 无              | 过渡自然           |

> 在设置页（齿轮图标进入）可用滑块图标开关声音/震动。

---

### 7. 统计系统（纯图形）

**入口**：顶部日历图标。  
**统计面板内容**：

- 顶部：三组环形图或横向柱形图，用颜色区分三种组件（绿/灰/黄），无标签，仅靠颜色和图例小色块指代。
- 中部：日历热力图，每个格子的深浅表示当日计数密度，一望即知。
- 底部：7天/30天趋势迷你折线图。
- 右上角分享图标：生成一张统计图片（含数字与图表，无隐私文字），可转发或保存。

_三件计数玩具，一个沉默的数字，一个让人上瘾的小小红点——这就是解压计数器的极简灵魂。_
